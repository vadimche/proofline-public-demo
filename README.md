# ProofLine public demo

A fictional, browser-only walkthrough of the ProofLine review workflow.

All people, employers, roles, scores, metrics, and outcomes are invented for
this demonstration. Nothing is uploaded, sent, saved, or generated by AI.
The score is illustrative and is not a hiring prediction.

## Local preview

```bash
npm --prefix gui ci
npm --prefix gui run build:public-demo
npm --prefix gui run preview
```

GitHub Pages publishes the static contents of `gui/dist` using the workflow in
`.github/workflows/pages.yml`.

## Public boundary

This repository intentionally contains no backend, profiles, job scans,
application records, credentials, accounts, uploads, persistence, analytics,
or network calls. It is a self-contained product demonstration.
