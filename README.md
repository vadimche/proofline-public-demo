# ProofLine public demo

A fictional, browser-only ProofLine application demo. It follows the product
workflow across Today, Top Jobs, Pack Studio, Application Readiness, Submit &
Track, and Profile & Evidence. Every interaction changes only in-memory demo
state for the current browser session.

All people, employers, roles, scores, metrics, and outcomes are invented for
this demonstration. Nothing is uploaded, sent, saved, or generated by AI.
The score is illustrative and is not a hiring prediction.

## Local preview

```bash
npm --prefix gui ci
npm --prefix gui run build:public-demo
npm --prefix gui run preview
```

GitHub Pages publishes the static contents of `gui/dist` using the workflow in
`.github/workflows/pages.yml`.

## Public boundary

This repository intentionally contains no backend, real profiles, real job
scans, real application records, credentials, accounts, uploads, persistence,
analytics, or network calls. It is a self-contained product demonstration
using invented JSON fixtures only.
